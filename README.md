# Hanaro 평가문제 - 프론트엔드 심화

## Getting Started

1. 개발자 모드 실행하기
   Run the development server:

```bash
pnpm dev
```

2. 프로덕션 모드 실행하기
   Run the build server:

```bash
pnpm build
pnpm start
```
